#

우리 금쪽이 정보를 블록체인에 저장하고 QR를 받급받으세요.

QR을 금쪽이 목걸이에 달아주세요.

금쪽이가 집을 나가면 QR로 신고하고 사례금을 등록하세요.

누군가 금쪽이를 발견하면 목걸이의 QR을 통해 주인에게 연락할거에요. 

금쪽이를 찾아준 은인은 사례금(ether)을 받습니다!
